Estructuras en C

Introducción

Una estructura en C es una colección de variables, potencialmente de diferentes tipos, agrupadas bajo un mismo nombre. Las estructuras son útiles para representar registros complejos, como los datos de un empleado, un punto en un espacio bidimensional, entre otros.

Sintaxis de una Estructura en C

La sintaxis básica de una estructura en C es la siguiente:

struct nombre_de_la_estructura { tipo_de_dato1 nombre_del_campo1; tipo_de_dato2 nombre_del_campo2; // más campos... };

Donde:

  • nombre_de_la_estructura: es el nombre que se asigna a la estructura.
  • tipo_de_dato1, tipo_de_dato2, ...: especifica el tipo de datos de cada campo de la estructura.
  • nombre_del_campo1, nombre_del_campo2, ...: son los nombres de los campos de la estructura.

Ejemplo de una Estructura en C

A continuación se muestra un ejemplo de cómo declarar e inicializar una estructura para almacenar la información de un empleado:

#include <stdio.h> #include <string.h> // Definición de la estructura struct Empleado { int id; char nombre[50]; float salario; }; int main() { // Declaración e inicialización de una variable de tipo struct Empleado struct Empleado empleado1; empleado1.id = 1; strcpy(empleado1.nombre, "Juan Pérez"); empleado1.salario = 3000.50; // Acceder e imprimir los campos de la estructura printf("ID: %d\n", empleado1.id); printf("Nombre: %s\n", empleado1.nombre); printf("Salario: %.2f\n", empleado1.salario); return 0; }

Acceso a los Campos de una Estructura

Los campos de una estructura se pueden acceder utilizando el operador punto (.). Por ejemplo, empleado1.nombre accede al campo nombre de la estructura empleado1.

Operaciones Comunes con Estructuras

  • Asignación: Se pueden asignar valores a los campos individuales de la estructura.
  • Lectura y Escritura: Se pueden leer y escribir valores en los campos de la estructura.
  • Inicialización: Las estructuras se pueden inicializar al declararlas o mediante asignaciones posteriores.
  • Pasar a Funciones: Se pueden pasar estructuras como argumentos a funciones, ya sea por valor o por referencia.
  • Arreglos de Estructuras: Se pueden crear arreglos de estructuras para manejar múltiples registros de datos similares.

Inicialización de Estructuras

Además de la inicialización por asignación individual de los campos, se pueden inicializar estructuras de manera directa al declararlas:

struct Empleado empleado2 = {2, "María López", 3500.75};

Pasar Estructuras a Funciones

Las estructuras se pueden pasar a funciones tanto por valor como por referencia. Pasarlas por valor implica que se hace una copia de toda la estructura, mientras que pasarlas por referencia permite modificar los campos de la estructura original.

// Función que recibe una estructura por valor void imprimirEmpleado(struct Empleado e) { printf("ID: %d\n", e.id); printf("Nombre: %s\n", e.nombre); printf("Salario: %.2f\n", e.salario); } // Función que recibe una estructura por referencia void aumentarSalario(struct Empleado *e, float incremento) { e->salario += incremento; } int main() { struct Empleado empleado3 = {3, "Carlos Sánchez", 4000.00}; // Llamada a la función que recibe la estructura por valor imprimirEmpleado(empleado3); // Llamada a la función que recibe la estructura por referencia aumentarSalario(&empleado3, 500.00); printf("Salario después del incremento: %.2f\n", empleado3.salario); return 0; }

Arreglos de Estructuras

Se pueden crear arreglos de estructuras para manejar múltiples registros. A continuación se muestra un ejemplo de cómo trabajar con un arreglo de estructuras:

struct Empleado empleados[3] = { {4, "Ana Torres", 3800.00}, {5, "Luis Ramírez", 4200.00}, {6, "Elena Díaz", 3700.00} }; for (int i = 0; i < 3; i++) { imprimirEmpleado(empleados[i]); }

Conclusión

Las estructuras son una herramienta poderosa en C para organizar y manipular datos complejos. Con una comprensión sólida de la sintaxis y las operaciones con estructuras, puedes escribir programas más efectivos y organizados, capaces de manejar datos de una manera coherente y estructurada.
